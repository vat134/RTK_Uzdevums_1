Dead by Daylight

This is one of the first projects I did for myself, the point of it was to help me do basic actions in the game without my participation and without implementing in the game files. 

The code analyzed my screen and every second looked for the pattern I gave it to press the button at the right moment to "catch the ball". My code itself caught the "ball" and did it with 88% accuracy. PYTHON language

Telegram

This project was done initially not for me, the task was simple to create a telegram bot that will scan the group and write all participants a message as if for news and advertising. The bot that I created coped with this goal and besides it recognized more information than necessary.

I stopped working on the project because the task was completed and it was lazy to customize all the options of dialogs and interaction with the bot. PYTHON language

Scripts for Dota 2

This is another script I wrote to practice programming and automate my game. The task was similar to that of Dead by Daylight without going inside the game to perform actions in it. The code I wrote had to press a lot of buttons in the right sequence in the minimum amount of time.

As a result, I was able to analyze my keyboard with the code and by pressing special keys to call the action of the code, which in turn itself pressed the right buttons did it quickly and even used the mouse itself. The code was written in PYTHON and even came in handy for my other friends.
